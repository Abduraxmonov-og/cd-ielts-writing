# Changelog

All notable changes to this project are documented here.

## [1.0.0] - 2026-05-17

### ✨ Initial Release

#### Features
- 📝 **Task 1 & Task 2** writing interface with real-time feedback
- ⏱️ **Timer** with exam countdown (60 min Task 1 + 2 min planning, 40 min Task 2)
- 📊 **Word Counter** with real-time tracking
- 🎯 **Practice & Exam Modes** toggle
- 🌙 **Dark Theme** with Sora + DM Mono fonts
- 🔍 **Task Selection** with filter chips and preview modal
- 📌 **Custom Task** always pinned first
- ⌨️ **Keyboard Shortcuts** (Ctrl+1, Ctrl+2 to switch tasks)
- 💾 **Full-screen** mode for distraction-free writing
- 📱 **Responsive Design** for desktop and tablet

#### Content
- 5 Task 1 examples (line_graph, bar_chart, pie_chart, map, process_diagram)
- 5 Task 2 examples (opinion, discussion, problem_solution, direct_question, advantages_disadvantages)
- Real Exam & Hot Topic badges

#### Architecture
- Clean separation: `selection.js` + `exam.js`
- Shared `style.css` design system
- JSON-based task bank (easy to extend)
- No dependencies—pure HTML/CSS/JS
